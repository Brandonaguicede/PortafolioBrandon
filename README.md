# Portafolio React estructurado

Proyecto creado con React + Vite + TypeScript + Tailwind CSS.

## Estructura principal

```txt
src/
  assets/
    photo.ts
  components/
    AnimatedCounter.tsx
    ProjectIcon.tsx
    SkillBar.tsx
  pages/
    Portfolio.tsx
  styles/
    global.css
  App.tsx
  main.tsx
```

## Instalación

```bash
npm install
npm run dev
```

## Compilar para producción

```bash
npm run build
npm run preview
```

## Notas

- El código original se dividió en componentes reutilizables.
- La imagen en base64 se movió a `src/assets/photo.ts`.
- La página principal quedó en `src/pages/Portfolio.tsx`.
